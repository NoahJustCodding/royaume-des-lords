## ⬇️ Installation

### Méthode rapide

1. Rendez-vous sur la page [**Releases**](https://github.com/NoahJustCodding/royaume-des-lords/releases/latest)
2. Téléchargez `royaume.des.lords.setup.exe`
3. Lancez l'installeur et suivez les étapes
4. Connectez-vous avec votre compte Microsoft
5. Choisissez un modpack et jouez !

> **Note :** Windows peut afficher un avertissement SmartScreen à la première installation.
> Cliquez sur **"Informations complémentaires"** puis **"Exécuter quand même"**.

## 🎮 Comment jouer

```
1. Lancez "Le Royaume des Lords"
2. Connectez-vous avec votre compte Microsoft (qui possède Minecraft Java Edition)
3. Dans "Modpacks Serveur", choisissez votre modpack
4. Cliquez "Télécharger" → attendez la fin du téléchargement
5. Cliquez "▶ Jouer" → Minecraft se lance automatiquement
6. Dans Multijoueur, le serveur est déjà configuré !
```
